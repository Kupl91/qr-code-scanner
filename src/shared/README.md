# Shared

Переиспользуемая инфраструктура.

## Структура

### api/
API клиенты и типы
- `box.ts` - API для работы с коробками
- `types/` - Общие типы данных
- `mocks/` - Моковые данные

### ui/
UI компоненты
- `button/` - Кнопки
- `input/` - Поля ввода
- `toast/` - Уведомления
- `card/` - Карточки

### lib/
Утилиты и хуки
- `hooks.ts` - Общие хуки
